Hue₃₂ CSS Polyfill

Every color on this page is written in Hue₃₂ notation. One script tag is all it takes.

<!-- Drop this in your <head> --> <script src="hue32-polyfill.js"></script> /* Then use hue32() anywhere in your CSS */ background-color: hue32(R23Y9, 24, 15); color: hue32(none, 2, 0); --brand: hue32(B32, 12, 20);

JavaScript API

The polyfill also exposes window.Hue32 for setting colors programmatically.

// Set a CSS custom property Hue32.set('--brand', 'R23Y9', 24, 15); // Convert to hex Hue32.toHex('B13M19', 12, 15); // '#4B0082' // Apply a color map Hue32.apply({ '--primary': ['B32', 12, 20], '--background': ['none', 31, 0], '--text': ['none', 2, 0], });
