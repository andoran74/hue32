The Notation

How to read a Hue₃₂ code

Every code is built from the same five axes. Once you know the pattern, any code is immediately readable.

H:R22Y10 S:24 T:15 L:0 F:0
H
Hue
Two RYGCBM letters with numbers summing to 32. R22Y10 = 22 parts red, 10 parts yellow. More of the first letter, leaning that direction. Pure colors use one letter: R32, B32.
S
Shade
Light to dark on a 1–32 scale. S:1 = black. S:16 = medium. S:32 = white. S:24 is medium-light.
T
Tone
Gray to vivid on a 1–32 scale. T:1 = gray. T:32 = fully saturated. T:0 = achromatic (no color information). T:15 is moderately vivid.
L
Luster
Surface finish from matte to glossy. L:1 = flat matte. L:32 = high gloss. L:0 = not encoded — digital color sources contain no surface information.
F
Fluorescence
UV-activated emission. F:1 = none. F:32 = intensely fluorescent. F:0 = not encoded. Relevant for paint, textiles, makeup, and security printing.
